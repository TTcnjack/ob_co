# Agentic Engineering Patterns

**来源**: https://simonwillison.net/guides/agentic-engineering-patterns/  
**作者**: Simon Willison  
**发布时间**: 2026年3月2日

## 概述

这是一份关于如何更好地使用编码代理(如 Claude Code 和 OpenAI Codex)的模式指南。

## 内容大纲

### 1. 原则 (Principles)
- **写代码现在很便宜** (Writing code is cheap now)
- **囤积你知道如何做的事情** (Hoard things you know how to do)

### 2. 测试和质量保证 (Testing and QA)
- **红/绿 TDD** (Red/green TDD)
- **首先运行测试** (First run the tests)

### 3. 理解代码 (Understanding code)
- **交互式解释** (Interactive explanations)
- **线性演练** (Linear walkthroughs)

### 4. 附录 (Appendix)
- **我使用的提示词** (Prompts I use)

## 核心思想

这份指南专注于在使用 AI 编码代理时的最佳实践模式,涵盖了从基本原则到具体的测试、代码理解方法,以及实用的提示词技巧。
